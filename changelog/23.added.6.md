- Added towncrier-based changelog management. Per-PR fragments under `changelog/` (`{PR}.added.md`, `{PR}.fixed.md`, ...) get rendered into `CHANGELOG.md` via the `generate-changelog` GitHub workflow. The repository's contributing notes describe the eight fragment types (`added` / `changed` / `deprecated` / `removed` / `fixed` / `security` / `performance` / `other`).
